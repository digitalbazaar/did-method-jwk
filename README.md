## Install

Requires Node.js 18+

To install from `npm`:

```
npm install --save @digitalbazaar/did-method-jwk
```

To install locally (for development):

```
git clone https://github.com/digitalbazaar/did-method-jwk.git
cd did-method-jwk
npm install
```
